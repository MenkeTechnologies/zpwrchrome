zpwrchrome

UNIX pass · Segmented Download Manager · Tab Switcher · Userscripts · History fzf
Version …

zpwrchrome is a browser power-tool by MenkeTechnologies — built around four pillars: a browserpass-compatible pass integration, a Chrono-inspired segmented download manager that takes over Chrome's default downloader, a JetBrains-style MRU tab switcher with scenes + opener-tree + minimap, and a Tampermonkey-equivalent userscript host with a GM_* shim.

The download-manager surface (this Settings UI, the queue manager, Rule System, Extension Filter) is a homage to Chrono Download Manager — reimplemented for Manifest V3 with a native Rust segmented downloader (browserpass-host-rs) that handles HTTP Range requests, cookie / user-agent forwarding, detached worker processes, and a file-state model surviving SW restarts.

Source / Issues: github.com/MenkeTechnologies/zpwrchrome
Native host crate: crates.io/crates/browserpass-host-rs
