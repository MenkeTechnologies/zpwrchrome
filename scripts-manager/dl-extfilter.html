Extension Filter

Buckets that classify finished downloads into Image / Video / Audio / Document / Archive / Application. Edit any cell to retune. The Rule System (next page) references these by Internal name.

Enable Display name Internal name File extensions

* Type extensions as a pipe-separated list — e.g. png|jpg|gif.
* Double-click any cell to edit; clear the row to remove it on save.
* ext_all is a meta-bucket — its extensions field is auto-derived from the others.
