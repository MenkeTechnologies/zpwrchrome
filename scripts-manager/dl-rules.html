Rule System

Each rule combines a condition over a job's fields with an optional naming mask. Conditions can reference Extension Filter buckets in [ext_*] form. Active rules drive the sidebar categories on the manager page.

Active As Tasks Filter Display name Internal name Condition Naming mask

* Examples: *time_add* >= [3_days_ago], *state* == [finished], {r_done} && *ext*.is([ext_image]), *size* > 0 && *size* < [kb(1024)].
