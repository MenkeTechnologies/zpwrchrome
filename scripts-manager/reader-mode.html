The killer feature

Jane Doe· example.com· 2026-06-05

Reader mode strips the active page to its main article and re-renders it inside a fixed-position overlay with the strykelang HUD palette. The original DOM is untouched — the overlay sits on top.

How it works

The content script picks an article root via a three-step heuristic: largest <article>, then <main> / [role="main"], then the densest paragraph cluster outside noise containers. The clone gets sanitized (scripts / iframes / nav / forms / inline event handlers stripped) and rendered with the chosen typography.

The point isn't to fight the page, but to extract its idea.

Use the A−/A+ buttons in the top bar to bump font size live; the setting persists. Press Esc to close.
